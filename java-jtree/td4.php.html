Fichiers utiles pour démarrer

Ce qu'il faut faire pour ce TD

  1. Comprendre, compiler, exécuter le programme ci-dessus. Modifier le pour que lors de la sélection simple d'un item de l'arborescence, une zone de texte située en dessous de l'arbre affiche le chemin complet jusqu'à l'item sélectionné.
  2. Sélection multiple: modifier le programme pour afficher dans une JTextArea tous les chemins des éléments d'une sélection multiple.
  3. Modifier le programme pour que lors de l'expansion ou de la réduction d'un nœud, un message s'affiche dans une zone de texte.
  4. Modifier le programme pour que les nœuds qui ont des enfants soient représentés par un icône particulier quand ils sont fermés, un autre icône lorsqu'ils sont ouverts. Ajouter également un icône particulier pour identifier les nœuds qui sont des feuilles.
  5. Ajouter une barre d'outils composée de deux boutons:
    • un bouton ajout qui ajoute un noeud fils au noeud sélectionné et
    • un bouton suppression, qui supprime le noeud sélectionné.
  6. Ajouter un deuxième arbre-vue sur le même arbre-modèle.
  7. Modifier le programme pour obtenir une architecture MVC.
